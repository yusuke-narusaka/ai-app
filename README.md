# AI App

AIを使ったツールをブラウザから直接試せるアプリ集です。

このプロジェクトは **GitHub Pages で公開**しているため  
リポジトリをクローンせずにそのまま利用できます。

## Demo

ブラウザからすぐ試せます。

https://yusuke-narusaka.github.io/ai-app/

---

## Overview

AIを活用した小さなツールや実験的なアプリをまとめたプロジェクトです。

目的

- AIツールの実験
- 業務効率化ツールの開発
- 個人開発プロジェクト
- AIアプリのプロトタイプ

---

## Features

- ブラウザだけで動作
- インストール不要
- GitHub Pagesで公開
- シンプルな構成

---

## Usage

1. 下記ページを開く

https://yusuke-narusaka.github.io/ai-app/

2. 利用したいAIツールを選択

3. ブラウザ上でそのまま実行
